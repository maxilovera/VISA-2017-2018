Dataset seleccionado: Series de precios de recursos forestales Subsecretaría de Desarrollo Foresto Industrial

Series de precios de recursos de origen forestal, desde el año 1996, clasificados por especie: rollizos con destino aserrado, debobinado, pasta celulósica, tableros. Estos se dividen también por provincia y por región.

Para el trabajo se simplificó el dataset para graficar únicamente los valores por año / por provincia, sin especificar la especie


Se eligió este dataset ya que representa datos desde una fuente gubernamental de mi país asociado a una temática interesante para mi, que es el desempeño de los recursos naturales. Este dataset forma parte de una serie de datos abiertos expuestos por el gobierno de Argentina
